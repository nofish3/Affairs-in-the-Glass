# 杯中事

杯中事是一款面向鸡尾酒入门用户的微信小程序。它帮助用户在酒吧看懂酒单、根据当晚的口味偏好选择一杯酒，并在本机简单记录喝过的酒。

## 核心功能

- **选一杯**：按酸甜苦咸鲜辣、香气、酒感和口感推荐一杯酒，并解释推荐理由。
- **识一杯**：优先识别酒名；未识别酒名时分析一种或多种配料组合。
- **记一杯**：本地新增、编辑和删除饮酒记录，不登录、不上传。

## 技术方案

- 微信原生小程序
- TypeScript + WXML + WXSS
- 内置酒款与配料数据，无后端依赖
- Vitest 自动化测试
- 微信本地存储保存饮酒记录

## 数据准确性

第一版启用 40 款含酒精鸡尾酒。中文名、英文名和主要配料以 International Bartenders Association（IBA）官方条目为基线；酸甜苦、香气、酒感、口感与饮用描述属于杯中事编辑判断。

详见 [数据来源说明](docs/data-sources.md) 和 [`recipe-sources.ts`](miniprogram/data/recipe-sources.ts)。

## 本地开发

```powershell
npm install
npm run typecheck
npm test
npm run test:coverage
```

使用微信开发者工具导入仓库根目录。`project.config.json` 当前使用游客 AppID，仅适合本地编译；预览和上传前需替换为项目真实 AppID。

## 视觉预览

浏览器直接打开 [`preview/index.html`](preview/index.html)，可查看首页、偏好、推荐、识别和记录五类关键画面。该目录只用于视觉验收，不参与小程序运行。

## 隐私与内容边界

- 不申请位置、相册、相机、通讯录和用户资料权限。
- 不上传解析输入或饮酒记录。
- 记录只保存在当前设备，清理缓存或更换手机后可能丢失。
- 未成年人请勿饮酒，理性饮酒。
